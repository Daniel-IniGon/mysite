About Me

My name is Daniel Iniesta González and I am currently making a career switch from web development into the video game industry. I have been learning in a self-taught way Video Game Development since my first Global Game Jam back in 2019. They are the best events to learn something new and meet interesting people.

I am eager to learn new things and I am improving my skills day by day learning from others too. I am a team player and find it essential to get along with the people you work with. That is why I am always willing to help whoever could need my aid. I am an easygoing person and try to fit in with anyone. There is always something in common that can make you connect with others.

I use Unity3d and C# to develop games, but I have been using other languages and technologies in my professional career like PHP, MySQL, CSS, and others. I am used to work with Git to keep the track of my projects. I am currently collaborating in the development of the game Orbitria's Tales (Moamoai indie studio first title) implementing different systems and designing game mechanics.

Latest Projects

Play Orbitria's Tales

Orbitria's Tales - MoaMoai

A mixed game between a turn-based RPG and a rogue-lite.

Both your team and your enemies share the same energy to realize attacks and support abilities, so each turn is a fight against each other and also for the resources.

Jack of all trades.

  • Refactor and improve the abilities system, making the workflow about 40% faster to create and test them.
  • Built a tool to autogenerate a custom mesh on runtime to visualize the procedurally generated dungeon on a minimap.
  • Gameplay programming. Implement AI behaviors.
  • Design and implementation of different game mechanics.
Play WoWTFT

WoWTFT - An autochess with the WoW pets

I think this is what some of Blizzard's fans expected when auto-chess genre became so popular. So I created my own approach. This project does not mean to be a complete game, but a fun way to test the skills I have been learning so far.

  • Learnt how to adapt code and tools based on the target platform.
  • Made a workflow to import the WoW assets. Built prefabs with the imported models and animations.
  • Added unity tools like the animator, profiler and shader graph to my toolkit.
  • Use of different mathematical algorithms to place game elements correctly.
Play Kanji Practice

KanjiPractice - Demo of an app to practice Japanese Kanji

As a Japanese student, I thought that building this little app, both to test my current skills and learn Japanese was a great idea.

You can practice the order stroke of the kanjis, also learn the pictograms main meaning and their 'On' and 'Kun' readings. There are 3 common words that contain that Kanji too.

  • Worked with the UI system in unity. Learnt how to properly build responsive UI's. Especially for mobile devices.
  • Incremented my SOLID knowledge by implementing more robust software using different patterns.
  • Learnt about technologies like PDollar that helped me to implement the systems I needed.
